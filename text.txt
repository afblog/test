به کمک کامند زیر می توانیم سورس کد را به آدرسی که در ریموت ها است 
انتقال دهیم یا پوش کنیم

git push (نام ریموت) (نام برنچ) => git push origin master

برای ریمو کردن یک ریموت هم از کامند زیر استفاده میکنیم
git remote remove origin

برای خلاصه تر پوش کردن می توان از کامند زیر استفاده کرد به این صورت که هر چی بعد از
فلگ یو قرار میگیرد برای گیت شناخته میشود و می گوییم هر وقت هر چی قبل از فلگ یو است 
را اجرا کردیم تو هر چی بعد از یو بود را برای ما قرار بده
git push -u origin master
